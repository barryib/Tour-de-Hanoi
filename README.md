#Tour de Hanoi

Ce jeux est la reprise du vieux jeux [Tour de Hanoi](http://fr.wikipedia.org/wiki/Tours_de_Hano%C3%AF) en mode texte.

Malgrés l'ancienneté de ce problème et la multitude de solutions proposées, la TOUR DE HANOI reste un jeu de réflexion digne ce nom.

Cette version est un mini projet de classe (année 2006) pour la mise en pratique des structures de données en C.
